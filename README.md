# Mini User Management System

## Project Overview
A full-stack User Management System that supports authentication, role-based authorization, and basic user lifecycle management. Users can register, login, edit their profile, change password, and request admin access. Admins can view all users, activate/deactivate accounts, and manage user roles.

### Key Features
- User signup & login
- JWT-based cookie authentication
- Role-based access control (Admin/User)
- User profile management
- Admin dashboard with pagination
- Request admin access feature
- Responsive UI with Tailwind CSS
- Backend API hosted on Render/Railway
- Frontend hosted on Vercel/Netlify

## Tech Stack
- **Frontend:** React (Hooks) + Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** MongoDB (Atlas)
- **Authentication:** JWT (cookie-based)
- **Password Hashing:** bcrypt
- **Deployment:** Vercel (Frontend), Render/Railway (Backend)
